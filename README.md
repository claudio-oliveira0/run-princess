# Run, Princess!

Run, Princess! é um jogo de plataforma 2D feito na engine Godot.

O objetivo do jogo é chegar até a linha de chegada o mais rápido possível, coletando moedas e evitando as quedas!

O jogo possui um sistema de cronômetro e recorde, incentivando speedruns. Com o tempo, será adicionado um ranking com os melhores tempos e conquistas para jogadores que terminarem em menor tempo!

---

## 🎮 Controles

← / → → mover para esquerda e direita  
SPACE → pular  
E → atacar  
SHIFT → correr  
R → reiniciar fase rapidamente  

---

## 🕹 Mecânicas do jogo

- Movimento de personagem
- Ataque
- Baús quebráveis
- Moedas coletáveis
- Sistema de cronômetro
- Recorde de tempo
- Parallax de fundo
- Menu inicial
- Música e efeitos sonoros

---

## 🛠 Engine utilizada

Godot Engine
